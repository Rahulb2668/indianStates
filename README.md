## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

1. Clone the repository:

```bash
git clone <your-repo-url>
cd guess-indian-states
```

2. Install dependencies:

```bash
npm install
# or
yarn install
```

3. Run the development server:

```bash
npm run dev
# or
yarn dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment on Vercel

1. Push your code to GitHub
2. Connect your repository to Vercel
3. Deploy with one click!

Or use the Vercel CLI:

```bash
npm i -g vercel
vercel
```

## How to Play

1. Type the name of an Indian state in the input field
2. Press Submit or Enter
3. If correct, the state will appear on the map in green
4. If incorrect, you'll get a "Wrong! Try again" message
5. Try to guess all 28 states!

## States Included

The game includes all 28 Indian states:

- Andhra Pradesh, Arunachal Pradesh, Assam, Bihar
- Chhattisgarh, Goa, Gujarat, Haryana
- Himachal Pradesh, Jharkhand, Karnataka, Kerala
- Madhya Pradesh, Maharashtra, Manipur, Meghalaya
- Mizoram, Nagaland, Odisha, Punjab
- Rajasthan, Sikkim, Tamil Nadu, Telangana
- Tripura, Uttar Pradesh, Uttarakhand, West Bengal
